# 🎓 Student Exam Score Predictor

A Machine Learning web application that predicts student exam performance based on factors such as study hours, attendance percentage, mental health rating, sleep duration, and part-time job status.

## Features
- Real-time exam score prediction
- Interactive Streamlit dashboard
- Machine Learning model trained using student performance data
- User-friendly interface with instant feedback
- Online deployment via Streamlit Community Cloud

## Tech Stack
- Python
- Scikit-Learn
- Pandas
- NumPy
- Streamlit

## Input Parameters
- Study Hours Per Day
- Attendance Percentage
- Mental Health Rating
- Sleep Hours Per Night
- Part-Time Job Status

##  Output
Predicted exam score along with performance insights and recommendations.
